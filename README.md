# GenQR

About the Project
GenQR is a web application designed to provide an easy and efficient way for users to generate QR codes from URLs. The application features a simple and intuitive interface that allows users to quickly convert any valid URL into a scannable QR code. The project focuses on usability and performance, ensuring that the QR code generation process is fast, reliable, and accessible across different devices.

Features
URL to QR Code Conversion: Users can generate a QR code from any valid URL instantly.

Live QR Code Preview: A live preview of the generated QR code is displayed before users decide to download or use it.

QR Code History: The application automatically saves previously generated QR codes, along with their corresponding URLs and timestamps, for future reference.

Dark/Light Mode Toggle: A toggle option is provided to switch between light and dark themes, offering a better user experience in different lighting conditions.

Download Option: Generated QR codes can be easily downloaded as image files for offline use.

Responsive Design: The application is fully responsive, ensuring a seamless experience on mobile devices and desktops.

Fast and Lightweight: The app is optimized for speed and efficiency, ensuring quick QR code generation without unnecessary delays.
